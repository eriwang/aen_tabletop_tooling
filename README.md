# AEN Tabletop Tooling

Tooling for "DND" AEN.

## Dependencies

- NodeJS (Must be 14.x, that's what Cloud Functions supports)
- Yarn

## Set Up Dev Environment

- `yarn install`
- `yarn firebase login`
- Hopefully that's it!

See `package.json` for some useful commands (e.g. `yarn lint`, `yarn test`, `yarn build`, `yarn deploy`.....)
